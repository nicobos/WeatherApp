# WeatherApp

Weather application showcase.

## App technology stack and architecture 
   - AndroidX libraries
   - MVVM Architecture
   - Repository pattern
   - Viewbinding
   - Hilt dependancy injection 
   - Flows / Flow operators
   - Coroutines
   - Google play services (device location)
   - Retrofit/okhttp3
   - Recyclerview
   - Coil (Image loading)
   - Material 3 theme implementation
   - Mockito 
   - Roboelectric
